# 运算符优先级


### 优先级和结合性

```js filename="表达式"
a OP1 b OP2 c
```

左结合（左到右）相当于把左边的子表达式加上小括号 (a OP b) OP c，右结合（右到左）相当于 a OP (b OP c)

**赋值运算符是右结合的**

```js filename="赋值运算"
a = b = 5; // 相当于 a = (b = 5);
```

**无论结合性和优先级如何，求值顺序总是从左到右**

**只有幂运算符是右结合的，而其他算术运算符都是左结合的**

```js filename="幂运算符"
2 ** 3 ** 2 // 512 相当于 2 ** (3 ** 2)

2 ** 3 / 3 ** 2 // 2 相当于 (2 ** 3) / (3 ** 2)
```
